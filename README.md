O menu, voltado para a acessibilidade, consiste em:
- Jogar;
- Opções;
- Créditos;
- Sair.

### Opções para a acessibilidade
Dificuldade dinâmica/adaptativa é parte essencial do jogo, portanto não será opcional.
- Volume;
    - SFX;
    - Música;
- Filtros de cor;
- Alto-contraste;
